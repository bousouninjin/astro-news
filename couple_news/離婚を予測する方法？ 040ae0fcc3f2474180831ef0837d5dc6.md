# 離婚を予測する方法？

![image.png](%E9%9B%A2%E5%A9%9A%E3%82%92%E4%BA%88%E6%B8%AC%E3%81%99%E3%82%8B%E6%96%B9%E6%B3%95%EF%BC%9F/image.png)

DaraとOliverは、Gottman博士が用意した「ラブラボ」にて向かい合っています。20代後半の彼らは、新婚夫婦を対象とした研究に参加することを志願しました。この大規模な研究では、130組のカップルが自分たちの結婚生活を顕微鏡で観察されるだけでなく、カメラの前でも検証することに同意しています。DaraとOliverは、「ラブラボ」のアパートで一泊観察される50組のカップルの中の一組です。

DaraとOliverは、自分たちの生活は忙しいけれど幸せだと話します。彼女は夜間に看護学校に通い、彼はソフトウェアエンジニアとして長時間働いています。多くのカップルと同様に、幸せを保っているカップルもいれば、最終的に離婚するカップルもいますが、DaraとOliverも自分たちの結婚が完璧ではないと認識しています。しかし、彼らはお互いを愛し合い、共にいることにコミットしていると語ります。二人が将来築きたい生活について話すとき、彼らの顔には幸せな笑顔が広がります。

## **議論の時間：ラブラボでの15分間**

私はDaraとOliverに、現在抱えている問題について15分間話し合いをするようお願いしました。この間、彼らの様子をビデオ撮影しながら観察します。会話中、彼らの体に取り付けられたセンサーが、心拍数などの循環系の様々な測定に基づいて、ストレスレベルを計測します。

当然ながら、私は彼らの話し合いが少なくとも多少のネガティブな要素を含むだろうと予想しました。結局、私は彼らに「言い争う」よう依頼したのです。理解のある言葉や笑顔で問題を解決するカップルもいますが、多くの場合、緊張感が漂います。DaraとOliverも例外ではありません。Daraは、Oliverが家事を十分に手伝っていないと感じており、Oliverは彼女が自分をあまりにもせっつきすぎると感じていて、そのためにやる気を失っていると話します。

## **結婚生活の未来を予測する**

彼らのこの問題についての会話を聞いた後、私は悲しいことに、DaraとOliverの結婚生活が徐々に不幸になるだろうと予測しました。案の定、4年後、彼らは離婚の危機に直面していると報告してきました。まだ一緒に住んでいるものの、彼らの生活は孤独なものとなり、かつて彼らを生き生きとさせた結婚生活はまるで幽霊のように存在しているだけです。

私が彼らの結婚生活が崩れるだろうと予測した理由は、彼らが言い争ったからではありません。結局のところ、私は彼らに議論を依頼したのです。夫婦間の怒りそのものが、結婚の崩壊を予測するわけではありません。同じ新婚夫婦の研究で、DaraとOliverよりもはるかに激しく言い争ったカップルがいます。しかし、多くのこれらのカップルは幸せに結婚生活を続けるだろうと予測しており、実際にそうなっています。DaraとOliverの未来の別れの手がかりは、彼らがどのように言い争うかにあり、それが増加するネガティブさと不信感を生むのです。

## **最初のサイン：厳しい議論の始まり(Harsh Start up)**

このディスカッションまたは議論（そしてこの結婚生活）がうまくいかない最も明らかな兆候は、その**始まり方です。**Daraはすぐにネガティブで批判的な態度を取ります。Oliverが家事の話題を持ち出すと、彼女はすぐに皮肉を言う準備をしています。「家事が足りてないってことね」と言います。Oliverはジョークで雰囲気を和らげようとします。「男は豚だって本を書こうか？」と。しかし、Daraは無表情のままです。

彼らは少し話を続け、Oliverが自分の役割を果たすための計画を立てようとしますが、Daraは「解決できたらいいけど、そうはならないみたい。リストを作ってみたけど、それもうまくいかなかったし、あなたに任せても、1か月何も進まなかった。」と話します。今や彼女はOliverを非難しています。つまり、問題は家事ではなく、彼自身であると言っているのです。Daraが夫に深い不満を抱いている理由は正当かもしれません。しかし、彼女の表現方法が、二人の違いを解決する上で大きな障害となります。

**議論が批判や皮肉（これも軽蔑の一形態です）から始まる場合、それは「厳しい議論の始まり」と呼ばれるものです。**Daraが非常に穏やかな声で話しているにもかかわらず、彼女の言葉には強いネガティブな力が込められています。彼らの会話の最初の1分間を聞いた時点で、彼らが違いを全く解決できていないことに私は驚きません。研究によると、議論が厳しいスタートアップで始まると、それは必然的にネガティブな結末に終わります。途中で「仲直り」の試みがいくらあっても、その結果を覆すことはできません。統計が物語るように、15分間の対話のうち、最初の3分間で話し合いの結果を96％の確率で予測できるのです。厳しい議論の始まりは、失敗を避けられません。ですので、**もしそのように議論が始まった場合、いっそ中断して、一度冷静になってからやり直すべきでしょう。**

## 議論が始まったらどうするべき？

1. **議論の開始**
    - 何らかの問題について話し合う。
2. **ネガティブな態度の確認**
    - 批判や皮肉や軽蔑が含まれているか確認。
3. **議論の中断**
    - ネガティブな態度が確認された場合、議論を中断。
4. **冷静になる時間を取る**
    - お互いに冷静になる時間を設ける。
